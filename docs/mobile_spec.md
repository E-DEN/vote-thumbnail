# モバイル版 UI 仕様書

> **注意**: 本仕様書に記載されている px・z-index・カラーコード等の数値はすべて**モックからの参考値**です。
> 実装時は既存PCコードの定義（変数・スケール・カラーパレット等）に合わせて調整してください。
>
> **コンポーネント方針**: 各UI要素はPC版と**同一のHTML/CSS/JSコンポーネントを最大限流用**する。
> モバイル専用の差分はレイアウト・サイズ調整のみにとどめ、独自実装は必要最小限にする。

## 前提・技術方針

- **対象デバイス**: iPhone 15 Pro で動作確認（393px幅）。主流デバイスで崩れないこと
  - 想定幅: 360px（Android普及帯） 〜 430px（iPhone Plus/Max）
  - 固定px指定は避け、`%` / `vw` / `grid` / `flex` でリキッドレイアウトにする
- **実装方針**: バニラJS・HTMLファイル分離（`/mobile/index.html`）
- **レスポンシブ**: PC版とコード共有しない。モバイル専用ファイルとして実装
- **ノッチ・Dynamic Island対応**: CSS環境変数 `env(safe-area-inset-*)` を使用

```html
<meta name="viewport" content="width=device-width, initial-scale=1, viewport-fit=cover">
```

```css
.header    { padding-top: env(safe-area-inset-top); }
.bottom-nav { padding-bottom: env(safe-area-inset-bottom); }
```

## 画面構造（全タブ共通）

```text
┌─────────────────────────────┐  ← スマホ枠（最前面）
│ [チャンネル名 ▽]    ヘッダー │  ← ヘッダー（safe-area-inset-top考慮）
├─────────────────────────────┤
│                             │
│       メインコンテンツ       │
│        （タブごとに異なる）  │
│                             │
├─────────────────────────────┤
│ [一覧] [投票] [ランキング] [リアクション] │  ← ボトムナビ（最前面）

```

### Z-index 優先順位（上が手前）

| 要素 | z-index |
| --- | --- |

## チャンネル選択パネル

- **開き方**: ヘッダーの `チャンネル名 ▽` ボタンをタップ（`▽` でパネル開閉を示す）
- **動作**: 左からスライドイン（`transform: translateX(-100%)` → `translateX(0)`）
- **背景**: メインコンテンツに半透明オーバーレイ（`rgba(0,0,0,0.3)`）
- **閉じ方**: オーバーレイ部分タップ or パネル外タップ
- **全タブから開閉可能**
- **カードタップ**: 当該チャンネルに切替えてパネルを閉じる
- **カード内 `↗` アイコンタップ**: 対応するYouTubeチャンネルURLに遷移（PC版のチャンネル名クリックと同等）

### パネルスタイル（案B採用）

アイコン＋横長短冊ラベル形式（PCのワイドサイドバーと統一）

```text
┌──────────────────────┐
│ ┌────────────────┐   │
│ │ 🔴  チャンネルA ↗│ ← 選択中（背景グレー＋左端インジケーター）
│ └──────────────────┘   │
│ ┌──────────────────┐   │
│ │ 🔵  チャンネルB ↗│   │
│ └──────────────────┘   │
│ ┌──────────────────┐   │
│ │ 🟢  チャンネルC ↗│   │
│ └────────────────┘   │
└──────────────────────┘
   パネル幅: 220px
```

- 選択中カード: `background: #e8e8e8` + 左端に `4px` の黒バー（インジケーター）
- 非選択カード: `background: none` + ボーダーのみ
- カード: `border-radius: 8px`、高さ `54px`
- アイコン: 円形 `40x40px`、カード左端に配置
- テキスト: `font-size: 14px`、アイコン右に配置

## 各タブのレイアウト

### Tab1 — 一覧

- **レイアウト**: 2列グリッド（`display: grid; grid-template-columns: 1fr 1fr`）
- サムネイル幅: 約 `160px`（393px ÷ 2 - padding）
- サムネイル下にタイトルテキスト（2行まで）
- 縦スクロール、最終行が少しはみ出してスクロール感を演出

```text
┌──────┐ ┌──────┐
│サムネ│ │サムネ│
└──────┘ └──────┘
 タイトル  タイトル
┌──────┐ ┌──────┐
│サムネ│ │サムネ│  ← 最下段は少しナビにかかって見切れる
```

---

### Tab2 — 投票

- **レイアウト**: A/B サムネイルを縦に並べる（PC版は横並び）
- 各サムネイルは全幅（`width: 100%`）
- AとBの間に `VS` ラベル（絶対配置、中央）

```text
┌─────────────────┐
│        A        │
└─────────────────┘
         VS
┌─────────────────┐
│        B        │
└─────────────────┘
```

---

### Tab3 — ランキング

- **レイアウト**: 1列リスト（ニコニコ動画風）
- Tab1（2列グリッド）と差別化

各アイテム構造:

```text
┌──[順位]────────┐  タイトル2行
│  サムネイル    │  スコア・再生数
└───────────────┘
```

- サムネイル: `140x80px`（16:9より少し小さめ）
- **順位バッジ**: サムネイル左上角にオーバーレイ（`position: absolute; top: 0; left: 0`）
  - サイズ: `20x20px`、白背景・黒枠・太字
  - 最大3桁（100位まで）対応
- タイトル: サムネイル右に2行、`font-size: 11px`
- スコア: タイトル下に1行

---

### Tab4 — リアクション

- **レイアウト**: 動画上部固定 + 再生リスト縦スクロール

#### 動画プレイヤー

- サイズ: 全幅 × 16:9 = `393 × 221px`
- `position: sticky; top: header-height` でスクロール中も固定（検討）
- **シークバーは動画枠内に収める**（`position: absolute; bottom: 8px`）
  - バー: `height: 4px`、グレー背景 + 黒プログレス
  - ノブ: `12x12px` 円形

#### 再生リスト（動画下）

- PC版の右サイドバーをそのまま下に移動（縦スクロール）
- 各アイテム: サムネイル（`130x73px`、16:9）+ タイトル2行（右側）
- 最終アイテムがボトムナビで少し切れてスクロール感を演出

```text
┌─────────────────────┐
│   16:9 動画         │
│              ──●──  │ ← シークバー（枠内）
├─────────────────────┤
│ ┌──────┐ タイトル  │
│ │サムネ│ 2行テキスト│ ↕ スクロール
│ └──────┘            │
│ ┌──────┐ タイトル  │
│ │サムネ│ 2行テキスト│
```
